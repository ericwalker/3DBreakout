# 3DBreakout
3D Breakout is an arcade game developed using Unity 5. Like the other break out game, use your paddle to avoid the ball drop down and destroy all of the bricks to win the game!


CONTROLS:

"Left" and "Right" Arrow keys for direction

'Space' for start

'Esc' for pause
